selectors in css

body {} is a selector.

a[href="https://google.com"] only works on tags with href as defined.

a[href*="google"] works on the ones that their href has google in it.

a[href^="https://"] works on the ones that start with the mentioned address.

a[href$=".com"] works on the ones that end with .com .

you can combine all these:a[href^="https://"][href$=".com"]

#product p: find any element that has product element and find paragraphs whithin that.

#product > p: find any element that has product element and find paragraphs whithin that which are the first child in hierarchy.

#product + p: any paragraph that is after the product tag will have the css attributes.

#product ~ p: all paragraphs after the product.
